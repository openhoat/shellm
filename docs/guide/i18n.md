# Internationalization

Termaid supports 7 languages for both the user interface and AI responses.

## Supported Languages

| Language | Code | Native Name |
|----------|------|-------------|
| English | `en` | English |
| French | `fr` | Français |
| Spanish | `es` | Español |
| German | `de` | Deutsch |
| Portuguese | `pt` | Português |
| Chinese | `zh` | 中文 |
| Japanese | `ja` | 日本語 |

## Language Settings

Termaid has two separate language settings:

### UI Language

Controls: Menu labels, buttons, error messages, tooltips, settings labels.

**Configuration**: Auto-detected from browser/system language. Change via the Language Selector in the configuration panel.

### AI Response Language

Controls: Language of AI-generated explanations, command interpretations, chat responses.

**Options**:
- `auto`: Use the UI language
- `en`, `fr`, `es`, `de`, `pt`, `zh`, `ja`: Force specific language

**Example**: You can have the UI in English while receiving AI responses in French.

## Auto-Detection

The system uses `i18next-browser-languagedetector` with this detection order:
1. LocalStorage preference
2. Browser language (navigator.language)

## Changing Languages

Languages can be changed without restarting:
1. Open the configuration panel (gear icon)
2. Select "Language" dropdown for UI
3. Select "Chat Language" for AI responses
4. Changes take effect immediately

## Contributing Translations

Translation files are located in `/src/locales/*.json`. To add or improve translations:
1. Edit the corresponding JSON file
2. Follow the existing structure
3. Submit a pull request

## Fallback Language

If your system language is not supported, Termaid falls back to English.
