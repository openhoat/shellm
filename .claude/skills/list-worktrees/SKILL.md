# Skill: List Worktrees

## Description

List all git worktrees and their current branches.

## Usage

```
/list-worktrees
```

## Example

```
/list-worktrees
```

## What It Does

Shows all worktrees with their paths and branches:
- Main worktree: `/home/openhoat/work/termaid` (main branch)
- Feature worktrees: `/home/openhoat/work/termaid-<feature>` (corresponding branches)
